STACK TECNOLOGICO ATTUALE
════════════════════════════════════════════════════════════════

Frontend:   Next.js 15 (React 18 + TypeScript + Turbopack)
Backend:    FastAPI (Python 3.11+) + Uvicorn
Database:   SQLite 3 (con Alembic + migrazioni manuali)
AI:         OpenAI GPT-4.1 / GPT-4.1-mini (API REST)
Meteo:      Open-Meteo API (gratuita, no API key)
Auth:       JWT + bcrypt
PWA:        Manifest.json + Service Worker
Rete:       Tailscale VPN per accesso remoto sicuro
CSS:        Vanilla CSS (design system custom, dark mode)
Deploy:     Windows (locale) — nessun cloud attivo
Porta FE:   3000 (Next.js dev server)
Porta BE:   8000 (Uvicorn, host 0.0.0.0)
